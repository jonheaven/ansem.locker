# Product vision

ansem.locker is the **diamond hooves** layer for the $ANSEM (Black Bull) community on Solana.

## Problem

Holders want to **prove conviction** publicly — not just claim "diamond hands" on X, but show **on-chain** evidence they cannot sell until a chosen date.

## Solution

Voluntary SPL token timelock:
- User deposits $ANSEM into program vault
- Solana enforces unlock time
- Shareable proof + public leaderboard

## Core loop

```
Connect wallet → Lock $ANSEM → Post to X → Others join → Leaderboard competition
```

## Differentiation vs ansem.info

ansem.info tracks airdrops passively. ansem.locker is **active commitment** — users lock real tokens and compete on conviction.

## Trust principles

1. Open source from minute one
2. Non-custodial — program holds tokens, not us
3. Doxxed builder with verifiable history
4. Small scope program — auditable LOC count
5. No seed phrases, ever
